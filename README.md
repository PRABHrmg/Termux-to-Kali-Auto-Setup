# 🔥 Termux → Kali Auto Installer (RMG)

> One-click installer for Kali Linux (NetHunter rootless) on Termux with Full / Minimal / Slim options

---

## 🚀 Features

* ✅ One-click setup
* 📦 Install Kali Linux (NetHunter rootless)
* ⚡ Choose installation type:

  * Full (~6GB)
  * Minimal (~2GB)
  * Slim (~1GB)
* 🔧 Automatic dependency installation
* 🎯 Beginner friendly

---

## 📥 Installation

```bash
git clone https://github.com/PRABHrmg/Termux-to-Kali-Auto-Setup.git
cd Termux-to-Kali-Auto-Setup
chmod +x setup.sh
bash setup.sh
```

---

## 🧠 How It Works

This script acts as a wrapper around the official Kali NetHunter installer.
It automatically:

* Updates Termux packages
* Installs required dependencies (wget, proot, git)
* Downloads the NetHunter installer
* Lets you select installation type
* Starts the installation

---

## 📦 Installation Options

| Option  | Size | Description         |
| ------- | ---- | ------------------- |
| Full    | ~6GB | Complete Kali tools |
| Minimal | ~2GB | Basic tools only    |
| Slim    | ~1GB | Lightweight version |

---

## ▶️ Start Kali

After installation, run:

```bash
nethunter
```

---

## ⚠️ Requirements

* Android device with Termux
* Stable internet connection
* Minimum storage:

  * 2GB (Slim)
  * 4GB (Minimal)
  * 8GB (Full recommended)
